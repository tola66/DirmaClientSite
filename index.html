Frequently Asked Questions

How does key activation work?

+

Enter your activation key on the website or in DirmaLoader.exe. The key will be verified on the server, removed from the available keys list, and your subscription will be activated permanently. Your HWID will be bound on first client launch.

Is Java required for the client?

+

Yes, the loader requires Java 25 or higher. You can download it from the link above or from Oracle's official website.

Is Dirma Client safe to use?

+

Yes, Dirma Client is completely safe. We use modern protection methods and do not collect users' personal data.

Which Minecraft versions are supported?

+

Currently supports Minecraft 1.16.5. Support for other versions is planned in future updates.

How to update the client?

+

The loader automatically checks for updates on startup. You can also check for updates manually in settings.
